# 🎃 The Curse of Skip (v1.0)

¡Bienvenido a **The Curse of Skip**! Un juego de supervivencia y gestión de cámaras donde tu velocidad de reacción lo es todo. ¿Podrás sobrevivir a lo que acecha en la oscuridad? 👁️👁️

---

## 🕹️ Instrucciones de Juego

Para mantenerte a salvo y salir con vida, domina los siguientes controles:

*   **Teclas `0` al `9`:** Cambia entre las diferentes cámaras de seguridad para monitorear las instalaciones.
*   **Barra Espaciadora (`Espacio`):** Cuando un monstruo logre colarse en la **Cámara 0**, presiónala inmediatamente para ahuyentarlo antes de que sea demasiado tarde.
